Etude#1 by Dr.C

Generative Audio-Visual Composition

Etude#1 is an autonomous generative composition system that creates an ever-evolving soundscape in the B Phrygian mode. Combining real-time synthesis via Csound with audio-reactive 3D particle visualizations, this piece explores the intersection of algorithmic composition, immersive sound design, and interactive visual art.

Each performance is unique—fourteen independent instrumental layers interact through probabilistic decision-making, creating emergent musical structures that never repeat exactly. The particle system responds to the audio in real-time, with flocking behaviors creating organic, lifelike motion patterns synchronized to the music.

Features

  • 14 Autonomous Instrumental Layers: FM bells, marimba, drones, noise swells, rumble, chords, flute, cello, crickets, birds, water drips, and crackling fire
  • 10 Sophisticated Effects: Multiple reverbs, delays, shimmer, chorus, and spatial processing
  • Audio-Reactive Particles: 333-3333 particles with flocking behavior responding to live audio levels
  • Full Control: Individual layer volumes, mute/solo, particle count/shape, and visual backgrounds
  • Curated Visuals: 45 high-quality background images including deep space, planets, clouds, sunsets, and Earth landscapes
  • Stereo Widening: Haas effect spatial processing for immersive headphone listening
  • Persistent Settings: All preferences saved via localStorage

Technical Details

Synthesis Engine: Csound 6.18 (via WebAssembly)

Visual Engine: Three.js r128

Compositional Mode: B Phrygian (B C# D E F# G# A)

Sample Rate: 44.1kHz, 32-sample buffer, stereo output

Audio Architecture: Recursive autonomous composers with probabilistic note generation

Particle System: Flocking algorithm with cohesion, separation, and alignment forces

Performance: Client-side only—no server required, runs entirely in browser
